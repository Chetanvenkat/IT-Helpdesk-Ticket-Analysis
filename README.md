# IT Helpdesk Ticket Analysis Dashboard

## 📊 Project Overview
This project is a Power BI dashboard that analyzes IT helpdesk tickets to identify trends, performance, and key insights.

## 🛠 Tools Used
- Power BI
- DAX
- Excel / CSV dataset

## 📌 Features
- KPI cards (Total, Resolved, Pending, Avg Resolution Time)
- Ticket distribution by priority
- Issue type analysis
- Channel analysis
- Customer satisfaction insights
- Interactive slicers

## 📈 Insights
- Ticket distribution across priorities
- Most common issue types
- Resolution efficiency
- Customer satisfaction trends

## 📂 Files
- dashboard.pbix → Power BI file
- dataset.csv → Dataset used

## 🚀 Outcome
This dashboard helps analyze helpdesk performance and supports data-driven decision-making.
